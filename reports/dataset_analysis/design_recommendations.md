## Rationale
- **Sampling Frequency**: Use the dominant sampling frequency if all/most recordings match; otherwise keep frequency-aware feature extraction or resample.
- **Bandpass**: 0.5-40 Hz preserves delta through low-gamma features while reducing drift and high-frequency noise.
- **Notch**: 50 Hz notch is included because the requested pipeline targets 50 Hz line-noise suppression.
- **Windowing**: Recommendation favors a candidate with manageable class imbalance while keeping windows short enough for seizure localization.
- **Labeling**: Any-overlap labeling captures seizure boundary windows and matches the clinical goal of detecting seizure presence.
- **Channels**: A common-channel subset makes feature columns consistent for classical ML and patient-wise testing.
- **Validation**: Patient-wise holdout avoids leakage from correlated windows belonging to the same subject.

## Review Warnings
